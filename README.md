## Tech Stack

1. Language - Javascript
2. Frontend - ReactJs, Bootstrap, slick-slider
3. Backend - Nodejs, Expressjs, MongoDB

## Run Project

### Backend

1. `cd backend`
2. `Create .env file and add your MONGODB_URL`
3. `npm install`
4. `npm start`

### Frontend

1. `npm install`
2. `In .env file replace API_URL with http://localhost:5000`
3. `npm run dev`
